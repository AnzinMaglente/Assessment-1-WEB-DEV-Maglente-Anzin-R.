HTML Basics

<head> Tag

The head tag is where you will find all of the meta data about the website, stuff that you wouldn't see normally unless you go through the website's code. This can be the tab name, language when booting up, and external files.

Example:


<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link rel="stylesheet" href="style.css">
  <title>TitleofPage</title>
</head>

Output:


This adds a link to an external stylesheet, change the content based on the device width, add a title to the page, and more.

<a> Tag

The <a> Tag is used to create a hyperlink to another page/part of the website, the link is specified in the href attribute where if you click on the content it is on, it will lead you to somewhere else.

Example:


<a href="https://metaverseage.ae/">Click on me to go to the metaverse page.</a>

Output:


Click on me to go to the metaverse page.





HTML Tags

<button> Tag

The button tag defines a clickable button, it has several attributes: autofocus, disable, type, value.

Example:


<button type="button">This is a button</button>

Output:




<!--> Tag

The <!--> Tag is used to add comments in the code.

Example:


Hello, this is a normal line.
<!--This is a comment line-->

Output:


Hello, this is a normal line.





CSS basics

What are Selectors?

Selectors are patterns use to select and style HTML elements. There are more than 60 selectors and five combinators that can be used to provide specific rules for the elements.

Example:


<style>
    .sample {
        color: white;
        background-color: black;
    }
</style>

<div class="sample">Hello world</div>

Output:

Hello World


CSS properties

Height and Width properties

Height and Width is used to set the height and width of an element. This can be specified by pixels or percentages.

Example:


<style>
    .hw {
        height: 50px;
        width: 50%;
        background-color: purple;
    }
</style>

<div class="hw">Hello world</div>

Output:


This has a height of 40 pixels and a width of 50%


<color> property

The color property changes the color of text.

Example:


<style>
    .color-test {
        font-size: 20px;
        color: Red;
    }
</style>

<div class="color-test">I am Red!</div>

Output:

I am Red!


<Padding> property

The padding tag sets a specific spacing between the items within the element. You can specify if you want the padding to be left, right, top, or bottom. However, if you only indicate padding, it will apply the padding to all directions 

<style>
    .padding-test {
        color: white;
        background-color: red;
        padding: 30px;
    }
</style>

<div class="padding-test">I have 30 pixels of padding</div>

Output:

I have 30 pixels of padding

Heading Tags

Heading tags are defined as <h1> to <h6> tags. It helps change the font size of each text, h1 being the most important while h6 being the least important.

Example:


<h2>Hello world</h2>
<h3>Hello world</h3>
<h4>Hello world</h4>

Output:

Hello world

Hello world

Hello world



<img> Tag

img tags are used to add any image format (jpeg, jpg, png, etc) to the website for viewers to see, there are four attributes you can change in this tag. The source file (src), alternative text (alt), width, and height.

Example:


<img src="cat photo" alt="Meow" width=100px height=100px >

Output:






<br> Tag

The br tag or break tag inserts a single break line to the text.

Example:


Hello, I am Anzin Maglente
<br><br>
How are you?

Output:


Hello, I am Anzin Maglente
How are you?

<ul> Tag

The ul tag creates an unordered list with the il tag to make each item in the list.

Example:


<ul>
    <li>Yeast<\li>
    <li>Baking Soda<\li>
    <li>Flour<\li>
</ul>

Output:

  • Yeast
  • Baking Soda
  • Flour

<comment> Tag

The comment tags are used to explain pieces of code.

Example:


<style>
    p {
        color: red;
    }
    /* This is a single-line comment */
</style>

<p>Hello world</p>

Output:

Hello World





Font properties

There are several font tags that changes how text is displayed, font-family, font-size, font-style and more.

Example:


<style>
    .text-example {
        font-family: Tahoma;
        font-style: italic;
        font-size: 20px;
    }
</style>


<div class="text-example">Hello World</div>

Output:


Hello World


background-color property

background-color specifies the background color of an element.

Example:


<style>
    .backgroundcolor-example {
        color: white;
        background-color: blue;
    }
</style>


<div class="backgroundcolor-example">I am living in a blue world!</div>

Output:


I am living in a blue world!


Margin property

Margins are used to create space around the element outside it's defined borders, how they interact with other elements.

Example:


<style>
    .margin-example {
        margin: 20px;
    }
</style>


<div class="margin-example">Hello World</div>

Output:


I have a margin of 20 pixels
